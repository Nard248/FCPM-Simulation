    I/O API¶

    Auto-detection Loaders¶

    fcpm.io.load_director(source: Union[str, Path, np.ndarray, Dict[str, np.ndarray]], **kwargs) -> DirectorField ¶

    Load director field with automatic format detection.

    This is the recommended entry point for loading director fields. Automatically detects the input type and calls the appropriate loader.

    Parameters:

    Name Type Description Default
    source Union[str, Path, ndarray, Dict[str, ndarray]]

    Can be: - Path to .npz file (with nx, ny, nz keys) - Path to .npy file (4D array with shape (..., 3)) - Path to .mat file (MATLAB) - NumPy array (4D with shape (..., 3) or dict-like) - Dictionary with 'nx', 'ny', 'nz' keys

    		 required
    **kwargs

    Additional arguments passed to specific loaders.

    		 {}

    Returns:

    Type Description
    DirectorField

    DirectorField loaded from source.

    Examples:

    >>> # From file (auto-detect format)
>>> director = fcpm.load_director('director.npz')
>>> director = fcpm.load_director('data/director.mat')

    >>> # From numpy array
>>> director = fcpm.load_director(my_array)

    >>> # From dictionary
>>> director = fcpm.load_director({'nx': nx, 'ny': ny, 'nz': nz})

    fcpm.io.load_fcpm(source: Union[str, Path, Dict[float, np.ndarray], np.ndarray], angles: Optional[List[float]] = None, **kwargs) -> Dict[float, np.ndarray] ¶

    Load FCPM intensity data with automatic format detection.

    Parameters:

    Name Type Description Default
    source Union[str, Path, Dict[float, ndarray], ndarray]

    Can be: - Path to .npz file - Path to .tif/.tiff file - Path to directory containing image files - Dictionary mapping angle -> intensity (returns as-is) - 4D numpy array (angles, y, x, z)

    		 required
    angles Optional[List[float]]

    Polarization angles in radians (required for some formats). Default: [0, π/4, π/2, 3π/4]

    		 None
    **kwargs

    Additional arguments passed to specific loaders.

    		 {}

    Returns:

    Type Description
    Dict[float, ndarray]

    Dictionary mapping angle (float) to intensity array.

    Examples:

    >>> I_fcpm = fcpm.load_fcpm('intensities.npz')
>>> I_fcpm = fcpm.load_fcpm('data/fcpm_stack.tif', angles=[0, np.pi/4, np.pi/2])
>>> I_fcpm = fcpm.load_fcpm('data/images/', angles=[0, np.pi/4])

    Dict[str, Any]
